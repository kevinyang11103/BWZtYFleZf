# 仓库管理系统 java999

本项目是一款基于B/S架构的仓库管理系统，采用前后端分离的技术模式，实现企业管理链路的数字化。

## 技术栈

### 后端
- Spring Boot
- Spring Security
- MyBatis Plus
- Redis
- WebSocket

### 前端
- Vue
- ElementUI
- ECharts
- Vuex
- WebSocket

## 功能模块
- 客户管理
- 订单管理
- 生产管理
- 待检管理
- 入库管理
- 盘点管理
- 出库管理

## 系统角色
- 系统管理员
- 客户管理员
- 仓库管理员
- 生产管理员

## 运行环境
- 后端：Java 1.8+
- 前端：现代浏览器
- 数据库：MySQL 5.7+
- 缓存：Redis

## 部署步骤
1. 准备运行环境（Java、MySQL、Redis等）
2. 部署后端服务
3. 部署前端服务
4. 数据库初始化
5. 启动系统

## 目录结构
```
project/
├── backend/  # 后端代码
├── frontend/  # 前端代码
└── database/  # 数据库脚本
```

## 核心亮点
- 基于Spring Boot和Vue的双端分离架构，易于维护和扩展
- 引入Spring Security，提供企业级的安全保障
- 整合ECharts，实现数据可视化
- 严格遵循软件工程规范，确保系统的高质量和稳定性
